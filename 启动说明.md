# 🚀 CTF Blog 启动方式

## 方式一：直接打开（最简单）

双击 **`open-site.bat`** 直接用浏览器打开

> ⚠️ 注意：部分浏览器直接打开文件可能会有样式问题

---

## 方式二：静态服务器（推荐）

### 方法 A: 使用 PowerShell 脚本
右键点击 **`start-static.ps1`** → "使用 PowerShell 运行"

然后访问: http://localhost:8080

### 方法 B: 手动启动
```bash
# 进入项目目录
cd ctf-blog-vue

# 先确保已构建
npm run build

# 使用 Python 启动服务器（推荐）
cd dist
python -m http.server 8080

# 或使用 Node.js
cd dist
npx http-server -p 8080
```

然后访问: http://localhost:8080

---

## 方式三：开发模式（热更新）

```bash
cd ctf-blog-vue
npm run dev
```

访问: http://localhost:5173

---

## 🔧 常见问题

### 问题1: 端口被占用
修改 `start-static.ps1` 中的 `$port = 8080` 为其他端口，如 `8081`

### 问题2: 无法识别命令
- 确保已安装 Python 或 Node.js
- 或者直接用方式一打开

### 问题3: PowerShell 执行策略限制
以管理员身份运行 PowerShell，执行：
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
